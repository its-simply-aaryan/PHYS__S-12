ARDUINO PROGRAMMING: LOOPS

Here is a link to Nathan's tutorial on for() loops. With the help of this tutorial, I also learnt the application of a nested for() loop, which actually came pretty handy while doing this week's assignment.

I decided to make a circuit with 2 LEDs and two buttons. I also used 4 1K resistors to control the voltage but I did not want to make it straightforward. I made my program such that one LED is at HIGH when the button is not pressed and is at LOW when the button is pressed. It works the other way around with the second LED. Here is my Arduino Code: 

const int BUTTON1 = 2;
const int BUTTON2 = 4;
const int LED1 = 8;
const int LED2 = 12;
int BUTTONstate1 = 0;
int BUTTONstate2 = 0;

void setup()
{
  pinMode(BUTTON1, INPUT);
  pinMode(BUTTON2, INPUT);
  pinMode(LED1, OUTPUT);
  pinMode(LED2, OUTPUT);
}

void loop()
{
  BUTTONstate1 = digitalRead(BUTTON1);
  if (BUTTONstate1 == HIGH)
  {
    digitalWrite(LED1, HIGH);
  } 
  else{
    digitalWrite(LED1, LOW);
  }
  BUTTONstate2 = digitalRead(BUTTON2);
  if (BUTTONstate2 == HIGH)
  {
    digitalWrite(LED2, LOW);
  } 
  else{
    digitalWrite(LED2, HIGH);
  }
}

This week, we were tazsked with reading the Analog serials with a microcontroller. We were provided with a Adafruit Metro MO Express and I used that to Read my Analog serial. I made a very simple circuit using a potentiometer, which I had first drafted on Tinkercad as I was not fully comfortable working with circuits. I connected the potentiometer to ground and 5V power supply. I also connected to A0 pin to send serial communications to my microcontroller. I could see the Analog readings on my serial monitor by running the "AnalogReadSerial" program. It was also really interesting to see the values shifting on turning the potentiometer. Here is my code for this circuit:



// the setup routine runs once when you press reset:
void setup() {
  // initialize serial communication at 9600 bits per second:
  Serial.begin(9600);
}

// the loop routine runs over and over again forever:
void loop() {
  // read the input on analog pin 0:
  int sensorValue = analogRead(A0);
  // print out the value you read:
  Serial.println(sensorValue);
  delay(1);        // delay in between reads for stability
}
